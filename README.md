# Игра "StepFight"

### Пошаговая стратегия, в которой игроки на каждом ходу выбирают стратегию из 5-ти действий и сражаются!

#### Для установки зависимостей:
 `npm install`
#### Для сборки проекта:
 `npm run webpack`
#### Для запуска проекта:
 `npm start`

# [Попробовать](stepfighting.herokuapp.com)
