# Игра "StepFight"

### Пошаговая стратегия, в которой игроки на каждом ходу выбирают стратегию из одного удара и блока, и сражаются!

#### Для установки зависимостей:
 `npm install`
#### Для сборки проекта:
 `npm run webpack`
#### Для запуска проекта:
 `npm startGameProcess`

# [Попробовать](https://stepfighting.herokuapp.com)
